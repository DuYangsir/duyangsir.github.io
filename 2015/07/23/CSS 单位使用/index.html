CSS单位的使用

CSS的单位、宽高设置

概括：

常用的一些设置宽高的单位和属性

  1. em
  2. rem
  3. vw
  4. vh
  5. max-width
  6. max-height
  7. min-width
  8. min-height

em

它是描述相对于应用在当前元素的字体尺寸，相对于元素父级或自身，所以它也是相对长度单位。一般浏览器字体大小默认为16px，则2em == 32px，也可以重新定义：

body{
    font-size: 12px;
}
.div1{
    font-size: 2em;
}

则设置的“div1”中字体大小为24px。

rem

根元素（html）的字体尺寸，以（html）的字体尺寸大小来改变字体大小：

html{
    font-size: 12px;
}
.div1{
    font-size: 3em;
}

则设置的“div1”中字体大小为36px。

vw

viewpoint width，视窗宽度，1vw=视窗宽度的1%

.div1{
    width: 50vw;
}

vh

viewpoint height，视窗高度，1vh=视窗高度的1%

.div1{
    height: 50vh;
}

max-width

max-width属性设置元素的最大宽度

使用如下：

p
{
max-width: 200px;
}

也可以使用%（定义基于包含它的块级对象的百分比）来进行设置：

p
{
max-width: 90%;
}

max-height

max-height属性设置元素的最大高度

使用如下：

p
{
max-height: 200px;
}

也可以使用%（定义基于包含它的块级对象的百分比）来进行设置：

{
max-height: 90%;
}

min-width属性设置元素的最小宽度

使用如下：

min-width: 200px;
}

也可以使用%（定义基于包含它的块级对象的百分比）来进行设置：

{
min-width: 90%;
}

也可以使用%（定义基于包含它的块级对象的百分比）来进行设置：
